구글 시트 연결 방법 (2분, 한 번만)
  1. 새 구글 시트 만들기 (주소창에 sheets.new)
  2. 상단 메뉴 확장 프로그램 → Apps Script
  3. 기존 코드 다 지우고, 제가 따로 드린 연결 코드를 붙여넣고 저장
  4. 배포 → 새 배포 → 유형: 웹 앱, 액세스: 모든 사용자 → 배포 → 권한 허용
  5. 나온 웹 앱 URL을 복사해서 위 칸에 붙여넣기
연결하면 완료/보류를 누를 때마다 시트에 자동으로 쌓입니다. 안 해도 ‘엑셀로 받기’로 쓸 수 있어요.
